# Simulador Zeta-EGB

Simulador numérico para el manifiesto *"Stability Analysis of Metric Tensor Perturbations in Einstein-Gauss-Bonnet Gravity via Riemann Zeta Distribution"*.

## Estructura
- `src/`: Módulos de cálculo (zeta, entropía, QNM, Hawking, visualización)
- `notebooks/`: Cuaderno interactivo de demostración
- `figures/`: Gráficas generadas automáticamente

## Requisitos
- Python 3.12+
- Librerías: numpy, scipy, matplotlib, mpmath, jupyter
